How It Works
HMAC-SHA256 signed · 3-attempt retry with exponential backoff · S14
SIGN
X-Apex-Signature header
Every payload is HMAC-SHA256 signed with a per-webhook secret so receivers can verify authenticity.
RETRY
0s · 2s · 8s backoff
Up to 3 attempts per event before giving up; every attempt is logged to the delivery log.
SUBSCRIBE
Per-event subscriptions
Each webhook picks specific event types (budget_threshold, security, …) or "all events".
